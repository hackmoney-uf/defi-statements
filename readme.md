## DeFi Statements

### Frontend
All commands executed in `frontend` directory  

Install dependencies:
```shell
yarn
```
Run local:
```shell
yarn dev
```

### Contracts
All commands executed in `contract` directory  

Install dependencies:
```shell
yarn
```
Run tests:
```shell
yarn test
```
Compile:
```shell
yarn build
```
Deploy
```shell
yarn deploy --network [localhost|polygonMumbai]
```
